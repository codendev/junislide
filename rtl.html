meSlide

meSlide jQuery carousel plugin.

Download

Features

  • Supports right to left languages like Arabic and Persian.
  • Supports left to right languages like Arabic and Persian.
  • Supports elastic carousel.
  • Supports fixed width box carousel.
  • Can enable/disable pager.

Elastic

HOW TO MAKE AN ELASTIC SLIDER

Include the libraries and invoke the method.

<script src="js/jquery-1.8.1.js" ></script>
<script src="js/jquery.touchwipe.1.1.1.js" ></script>
<script src="js/meslide.js" ></script>


$(".meslide").meSlide(
 { 
   'image_width':245,
   'show':4,
   'image_space' : 3,
   'pager':true,
   'type':'elastic'   
 }
);
